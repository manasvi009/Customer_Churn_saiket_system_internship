# Customer Churn Analysis Dashboard

## Project Overview

This project focuses on analyzing customer churn behavior using Power BI. The dashboard helps identify factors contributing to customer attrition and provides actionable insights to improve customer retention.

## Objectives

* Analyze customer demographics and churn patterns.
* Identify key factors influencing customer churn.
* Measure churn rate and customer retention trends.
* Provide business insights through interactive visualizations.

## Tools & Technologies

* Power BI
* Microsoft Excel
* DAX (Data Analysis Expressions)

## Dashboard Pages

### 1. Executive Overview

* Total Customers
* Churned Customers
* Churn Rate %
* Revenue Analysis
* Customer Distribution

### 2. Customer Demographics Analysis

* Gender vs Churn
* Partner vs Churn
* Dependents vs Churn
* Senior Citizen vs Churn

### 3. Customer Tenure Analysis

* Customer Tenure Distribution
* Tenure vs Churn Analysis
* Average Tenure by Contract Type
* Churn Rate by Tenure Group

### 4. Churn Analysis

* Contract Type vs Churn
* Payment Method vs Churn
* Internet Service vs Churn
* Tech Support Impact on Churn
* Churn Rate Heatmap

## Key Insights

* Month-to-Month contract customers show the highest churn rate.
* Customers without Tech Support are more likely to churn.
* Long-tenure customers are less likely to leave.
* Electronic Check users have higher churn rates.
* Contract type significantly affects customer retention.
